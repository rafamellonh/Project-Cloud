# Azure Application Gateway

* O Azure Application Gateway é um balanceador de carga de aplicativo da Microsoft, que fornece vários recursos e benefícios para o gerenciamento, desempenho e segurança de aplicativos web. Abaixo alguns dos benefícios incluídos no Application Gateway

* Ele atua na camada 7 e faz o roteamento de carga entre os aplicativos, garantido a distribuição de carga equilibrada com base em algumas métricas (regras) assim podendo garantir a melhora do desempenho e a disponibilidade dos aplicativos
